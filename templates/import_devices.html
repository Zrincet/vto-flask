批量导入设备

上传JSON格式的设备配置文件，批量添加多个设备

使用说明
文件格式要求
  • JSON格式文件，UTF-8编码
  • 必须包含"devices"数组字段
  • 每个设备包含ip、section、building字段
  • position字段可选（位置信息）
导入规则
  • 重复IP地址的设备将被跳过
  • 自动生成设备名称和MQTT主题
  • 新导入设备默认为不可见状态
  • 导入后可在设备管理中调整
JSON格式示例 
{
  "description": "VTO门禁设备预设数据",
  "version": "1.0",
  "devices": [
    {
      "ip": "172.16.11.1",
      "section": "01",
      "building": "001", 
      "position": "1"
    },
    {
      "ip": "172.16.11.2",
      "section": "01",
      "building": "001",
      "position": "2"
    }
  ]
}
